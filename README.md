## エージェント系
ChatGPTに課金してるならCodexが使えるはず、ここに以下に示すプラグインを導入することで開発体験をより向上することができるはず
- [Ponytail](https://github.com/DietrichGebert/ponytail) ：コーディングエージェントにより簡潔で保守性に優れたコードを書かせる指示をするためのプラグイン。導入後、/ponytailをチャットで実行すること

## 画像系
- [FLUX.2](https://github.com/black-forest-labs/flux2)：画像合成に強い
- [Qwen-Image](https://github.com/QwenLM/Qwen-Image) ：リアル調の画像生成に強い

## 音声系
- [Irodori-TTS](https://github.com/Aratako/Irodori-TTS) ：ワンショットでのボイスクローン (10秒ほどのファイルからのボイス生成) が可能。絵文字による指示を踏まえた感情表現に優れる他、Voice-Designで存在しないボイスも作れる
- [Irodori-TTS=Server](https://github.com/Aratako/Irodori-TTS-Server) ：↑のリポジトリをサーバーで稼働させる
- [GPT-SoVITS](https://github.com/RVC-Boss/GPT-SoVITS) ：複数音声から学習させ、喋らせる事ができる　感情表現には弱いかも
- [Retrieval-based-Voice-Conversion-WebUI](https://github.com/RVC-Project/Retrieval-based-Voice-Conversion-WebUI) ：いわゆるRVC。現在は更新停止されているが、音声を学習した声に変換することができる
- [Applio](https://github.com/IAHispano/Applio) ：RVCフォーク。使いやすいようにいろいろな改善が行われている。
